
- Cambiar version ruby : rvm use 'version'

- Crear proyecto  : rails new --help

- Iniciar server : bin/rails server รณ bin/rails s, estando ubicado
en la carpeta del proyecto.
- rails s -h

- rails generate , nos sirve para crear generadores : model, controller, scaffold, etc

- Generar controlador : rails generate controller 'nombre' 'accion'

- Generar modelo : rails generate model Article title body:text visits_count:integer

- Abrir consola rails : rails console
